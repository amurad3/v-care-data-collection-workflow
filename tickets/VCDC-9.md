# VCDC-9 · Stage backup device for session

- **Type:** Task
- **Priority:** Medium
- **Status:** Ready for Session
- **Labels:** device, prep

## Description
Charge and configure a backup prototype device so the session can continue if the
primary device fails.

## Acceptance criteria
- [ ] Backup device is charged to at least 80%
- [ ] Same app/build as the primary device is installed
- [ ] Backup device ID is documented and labeled
- [ ] Login is verified on the backup device
